# Chronospire - Текстовая RPG Игра

## 📖 Описание:
Хроноспираль - это эпическая текстовая RPG, в которой вы играете за героя, стремящегося отомстить богам времени за уничтожение всего, что он любил. Вам предстоит подняться по Хроноспирали - лестнице, соединяющей миры и времена, чтобы достичь Престола Хроноса и свергнуть повелителей времени.

## 🎮 Основные возможности:
### 👤 Создание персонажа:
1. Выбор имени - создайте уникального героя

2. Система классов - три уникальных класса на выбор:

3. Воин - сильный и выносливый боец ближнего боя

4. Маг - могущественный заклинатель, владеющий магией времени

5. Ассасин - ловкий и хитрый воин, мастер уклонений

6. Начальные предметы - каждый класс получает уникальные стартовые предметы

7. Случайные характеристики - базовые параметры генерируются с учетом выбранной сложности


### ⚔️ Механика боя:
1. Пошаговая система - игрок и противник атакуют по очереди

2. Расчет урона - учитывает атаку атакующего и защиту цели

3. Варианты действий в бою:

4. Атака - стандартная атака с шансом критического удара

5. Использование предмета - зелья лечения или временные усиления

6. Уклонение - шанс избежать атаки противника

7. Осмотр противника - информация о характеристиках врага

8. Уникальные враги - каждый противник имеет особые характеристики и описание


### 📈 Система прокачки:
1. Опыт и уровни - получайте опыт за победу над врагами

2. Повышение уровня - при достижении нового уровня:

3. Увеличивается максимальное здоровье

4. Добавляются очки характеристик для распределения

5. Распределение характеристик - улучшайте:

6. Максимальное здоровье

7. Атаку

8. Защиту

9. Уклонение


### 🎒 Система инвентаря:
1. Поиск предметов - находите предметы в сундуках:

2. Зелья здоровья и силы

3. Оружие (мечи, посохи, кольца)

4. Броня (щиты, плащи)

5. Экипировка - предметы дают постоянные бонусы к характеристикам

6. Использование предметов - зелья можно использовать для мгновенного эффекта


### 🗺️ Система локаций:
#### 1. Нелинейное перемещение - 8 уникальных локаций:

1.1 Врата Хроноспирали

1.2 Зал Прошлого

1.3 Мост Судьбы

1.4 Сад Вечности

1.5 Чертоги Богов

1.6 Сокровищница Времени

1.7 Престол Хроноса

1.8 Измерение Забвения (скрытая локация)

#### 2. Типы локаций:

2.1 Бой - сражения с врагами

2.2 Сундук - нахождение предметов и золота

2.3 Отдых - восстановление здоровья

2.4 Финальный бой - битва с боссом

#### 3. Текстовые описания - каждая локация имеет уникальное атмосферное описание


### 🎯 Уровни сложности:
#### 1. Игра предлагает 5 уровней сложности, влияющих на баланс игры:

1.1 Лёгкая - для начинающих искателей приключений

1.2 Средняя - сбалансированный игровой опыт

1.3 Сложная - испытание для опытных воинов

1.4 Безумная - выживание на грани возможного

1.5 Невозможная - вызов для истинных мастеров (открывает доступ к скрытому боссу)


## 🏪 Дополнительные системы:
### Магазин:
1. Покупка предметов за золото

2. Широкий ассортимент: от зелий до легендарных артефактов

3. Экономическая стратегия - планируйте свои покупки

### Боссы:
1. Бог Хронос - финальный босс с эпическими диалогами

2. Первозданный Хаос - скрытый босс, доступный только на сложности "Невозможная"

### Эмоциональная составляющая:
1. Атмосферные описания локаций

2. Постепенное раскрытие сюжета

### 🚀 Установка и запуск:
##### Требования:
Python 3.6 или выше

Стандартные библиотеки Python (random, time, sys)

##### Запуск игры
```python
python chronospire.py
```

### 🎮 Управление:
#### 1. Игра использует текстовый интерфейс:

1.1 Ввод цифр для выбора опций

1.2 Следование инструкциям на экране

1.3 Стратегическое планирование действий


### 💡 Советы для игроков:
1. Выбирайте класс в соответствии со стилем игры - каждый класс имеет уникальные сильные стороны

2. Используйте предметы стратегически - зелья могут спасти вас в трудной битве

3. Исследуйте все локации - вы можете найти полезные предметы и дополнительное золото

4. Распределяйте очки характеристик с умом - балансируйте между здоровьем, атакой и защитой

5. На высокой сложности - тщательно планируйте каждое действие, ошибки могут быть фатальны


### 🏆 Достижения:
1. Победа над Хроносом - завершите основную сюжетную линию

2. Абсолютная победа - победите обоих боссов на сложности "Невозможная"

3. Мастер тактики - пройдите игру на сложности "Безумная" или выше

4. Коллекционер - соберите все уникальные предметы


### 🎭 Сюжет:
Давным-давно боги времени установили свое владычество над вселенной. Они играли судьбами смертных, как пешками в своей вечной игре. Но однажды они совершили роковую ошибку - уничтожили всё, что вы любили. Теперь вы поднимаетесь по Хроноспирали, чтобы отомстить и свергнуть повелителей времени.


#### *Судьба вселенной висит на волоске, и только вы можете ее спасти... Помните: время не прощает ошибок, и каждая секунда может стать последней.*
